    USUARIOS Y PERMISOS

    AÑADIENDO NUEVO USUARIO

    Crear nuevo: 

    adduser usertest

    Por si queremos eliminar la contraseña de un usario especifico: 

    sudo passwd -d usertest
    -- Esto nos ayudará a mejorar la experiencia de usuario al usar el comando sudo

    INICIAR SESION NUEVO USUARIO

    su usertest

    DAR PERMISOS ROOT A NUEVO USUARIO

    Esto permitirá que nuestro usuario nuevo pueda hacer uso de sudo para tener privilegios de superusuario. 

    usermod -aG sudo usertest

    -- Ahora cada vez que escribamos sudo nos pedirá una contraseña para realizar alguna acción de súper usuario y si no tenemos preestablecido una contraseña pues simplemente bastará con escribir sudo.

    MODIFICAR O AÑADIR CONTRASEÑA

    sudo passwd usertest

    ELIMINAR USUARIO

    sudo userdel usertest

    LISTAR USUARIOS EXISTENTES

    cat /etc/passwd | cut -d":" -f1

    Quitar PASSWD a SUDO

    sudo visudo

    En la parte inferior del archivo, agregue la siguiente línea: 

    $USER ALL=(ALL) NOPASSWD: ALL
    Y 
    %sudo ALL=(ALL) NOPASSWD: ALL
    -- Este fuincionó luego de agregar esta linea desde el usuario logeado (pueda funcionar en algunos casos) fonts: - https://askubuntu.com/questions/147241/execute-sudo-without-password - https://askubuntu.com/questions/334318/sudoers-file-enable-nopasswd-for-user-all-commands

    ?Este suepuestamente

    Sirve para dar permisos a un grupo 

    # Los miembros del grupo de administración pueden obtener privilegios de root
%admin ALL=(ALL) ALL
